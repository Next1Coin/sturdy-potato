Sturdy Potato Flight Simulator

Take off, navigate waypoints, and land safely using realistic-ish flight controls.

Flight Instruments

    Mission

    Reach all waypoints, then return and land at the runway.

    Controls

    • W / S: Pitch down/up
    • A / D: Roll left/right
    • Q / E: Yaw left/right
    • Arrow Up / Down: Throttle +/-
    • Space: Air brake
    • R: Reset flight

    Tip: Keep speed above 60 knots to avoid stalling. Smooth control inputs work best.
